# Teslo API

1. Clone project in git

2. Install dependencies
```
yarn install
```

3. Clonar archivo ```.env.template``` y renombrarlo a ```.env```

4. Cambiar las variables de entorno

5. Levantar la BD
```
docker-compose up -d
```

6. Ejecutar endpoint GET /api/seed

7. Levantar proyecto:
```yarn start:dev```
